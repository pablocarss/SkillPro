COMO CRIAR:
1. Abra o Microsoft Word ou LibreOffice Writer
2. Copie o texto abaixo
3. Cole no documento
4. Formate como desejar (fontes, cores, bordas, etc.)
5. Salve como .docx
6. Faça upload em Admin → Templates de Certificado

========================================

VARIÁVEIS DISPONÍVEIS:

{nome} - Nome completo do aluno
{cpf} - CPF do aluno
{curso} - Título do curso
{carga_horaria} - Duração do curso
{data} - Data de conclusão (formato: 30 de dezembro de 2025)
{nota} - Nota final (ex: 95.5)
{hash} - ID único do certificado (16 caracteres)

========================================

DICAS DE FORMATAÇÃO:

1. Use uma fonte elegante como:
   - Times New Roman
   - Garamond
   - Georgia

2. Tamanho de fonte sugerido:
   - Título: 24pt, negrito
   - Corpo: 12pt
   - Hash: 10pt, monoespaçado

3. Adicione bordas decorativas:
   - Design → Bordas de Página
   - Escolha um estilo elegante

4. Use papel A4 em orientação Paisagem:
   - Layout → Orientação → Paisagem

5. Adicione seu logo:
   - Inserir → Imagem
   - Posicione no topo centralizado

6. Use cores institucionais:
   - Configure as cores do texto
   - Mantenha profissional
